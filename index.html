Building-an-end-to-end-AWS-devsecops-CI-CD-pipeline-with-SAST-and-DAST-tools

This project gives a brief information on AWS DevSecOps CI/CD pipeline with SAST and DAST tools automates secure software delivery. Code is committed to a repository, triggering the pipeline. SAST scans the code for vulnerabilities during build. The artifact is deployed, and DAST tools test the running application for security issues. Continuous monitoring and feedback loops ensure security is integrated throughout the development lifecycle.
